# Simulasi Penjadwalan Social Link Persona 3 Reload

Program ini dibuat untuk tugas Makalah IF2211 Strategi Algoritma. Tujuannya adalah melakukan simulasi penjadwalan Social Link pada game Persona 3 Reload dengan pendekatan algoritma Program Dinamis.

## Cara Menjalankan Program

Program ini ditulis dengan bahasa Python murni dan tidak memerlukan library eksternal tambahan. Untuk menjalankannya, cukup buka terminal atau command prompt di dalam folder ini, lalu jalankan perintah berikut:

```bash
python main.py
```

## Struktur File

- `main.py`: File utama untuk menjalankan program dan menampilkan hasil ke terminal.
- `scheduler.py`: Berisi logika utama algoritma penjadwalan (Program Dinamis).
- `social_link_data.py`: Berisi data karakteristik, hari ketersediaan, dan syarat setiap Social Link.
- `config.py`: Konfigurasi dasar seperti kalender, hari libur, dan fungsi utilitas terkait hari.
- `output.py`: Berisi fungsi-fungsi untuk mencetak hasil dan ringkasan eksekusi ke layar.
